# Projet d'illustration d'une séance d'initiation à l'IA

## Contexte :

à partir d'un cours élaboré par un collègue dans le cadre de l'enseignement scientifique [que vous trouverez ici](https://nuage03.apps.education.fr/index.php/s/JEsrBC4462iSPo) nous avons envisagé une application qui illustre le fonctionnement de la recherche d'une pondération des segments des chiffres écrits avec 7 segments qui permette de trier les chiffres pairs et impairs

## Réalisation :

on utilise surtout tkinter et un Canvas sur lequel on développe les outils qui permette d'illustrer le processus d'ajustement des pondérations.

pyinstaller fait partie des __requirements__ uniquement pour créer un executable sous windows pour des facilités d'usage des collègues à qui cette appli est destinée.
